# 🎮 Game Store - C#

Game store management system developed in C#, focused on object-oriented programming concepts such as classes, methods, and data handling.

## 📌 About the project

This project simulates the basic operation of a game store, allowing users to manage products, control inventory, and perform sales-related operations.

It was developed to practice fundamental C# concepts and programming logic.

## 🚀 Features

- 📦 Game product registration
- 💰 Price definition
- 🎮 Platform identification by console
- 🔢 Inventory quantity control
- 📊 Total inventory value calculation
- 🧾 Product information display

## 🛠️ Technologies used

- C#
- .NET
- Object-Oriented Programming (OOP)

## 🧠 Concepts applied

- Classes and objects
- Constructors
- Methods
- Encapsulation
- Lists (`List<T>`)
- Console input and output

## 📂 Project structure

```txt
Game_Store_CSharp/
├── Program.cs
└── GameStore.Console.csproj
```

## ▶️ How to run

- Clone the repository;
- Open the project in an IDE such as Visual Studio or VS Code;
- Run:

```bash
dotnet run
```

## 💡 Usage example

The system allows users to:

- add products;
- list registered products;
- search products by name;
- remove products;
- update products;
- calculate the total inventory value.
